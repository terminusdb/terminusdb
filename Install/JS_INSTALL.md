# Installation

---

## Dependencies

The TerminusDB Javascript Client has no external dependencies. It comes bundled with the axios library for communicating over HTTP, but otherwise has no third party dependencies.

## Install from npm

Installation

TerminusDB Client can be used as either a Node.js module available through the npm registry

Installation is done using the npm install command:

Using npm:

```bash
$ npm install --save @terminusdb/terminusdb-client
```

Minified Script

Using cdn:

```html
<script src="https://unpkg.com/@terminusdb/terminusdb-client/dist/terminusdb-client.min.js"></script>
```

Downloading:

Download the terminusdb-client.min.js file from the /dist directory and save it to your location of choice, then:

```html
<script src="http://my.saved.location/terminusdb-client.min.js"></script>
```
